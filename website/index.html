Frequently Asked Questions

Can GardenerGrid work where connectivity is unstable?

Yes. Core workflows include local caching and retry/replay logic so important field actions are not lost when networks are intermittent.

How does market pricing data work?

The platform combines USDA public report intelligence with community-submitted local prices to improve local decision quality.

Can this be shipped to iPhone users?

Yes. iPhone-compatible artifacts are produced using the included macOS CI workflow with Apple signing secrets configured in GitHub Actions.
